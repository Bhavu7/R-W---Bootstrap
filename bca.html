Bachelor of Computer Applications (BCA)

About BCA

The Bachelor of Computer Applications (BCA) program at DNICA is a three-year undergraduate degree designed to equip students with foundational and advanced skills in computer applications, programming, and IT management. Affiliated with Sardar Patel University, the program emphasizes practical learning and industry-relevant skills to prepare students for careers in software development, IT consulting, and more.

BCA Subjects & Syllabus

The BCA curriculum is structured across six semesters, covering core computer science subjects, programming, and electives. Below is an overview of key subjects and their weightage in the curriculum.

Core Subjects

  • Programming in C: Introduction to C programming, data types, control structures.
  • Database Management Systems: SQL, relational databases, normalization.
  • Web Technologies: HTML, CSS, JavaScript, and frameworks.
  • Data Structures: Arrays, linked lists, stacks, queues, trees.
  • Operating Systems: Process management, memory management, file systems.
  • Software Engineering: SDLC, Agile methodologies, testing.

Syllabus Weightage

Category Credits Weightage (%)
Core Subjects 60 50%
