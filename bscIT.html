BSc Computer Applications & Information Technology

About BSc CA & IT

The BSc in Computer Applications & Information Technology at DNICA is a three-year undergraduate program focusing on advanced IT concepts, networking, and system administration. Affiliated with Sardar Patel University, it prepares students for dynamic careers in IT infrastructure, cybersecurity, and software solutions.

BSc CA & IT Subjects & Syllabus

The BSc CA & IT curriculum spans six semesters, integrating IT fundamentals, networking, and advanced technologies. Below is an overview of key subjects and their weightage.

Core Subjects

  • Computer Networks: LAN, WAN, network protocols, and security.
  • Cloud Computing: Cloud models, virtualization, AWS basics.
  • Cybersecurity: Cryptography, ethical hacking, security policies.
  • Database Administration: Oracle, MySQL, database optimization.
  • Programming in Python: Python syntax, data analysis, automation.
  • IT Infrastructure Management: Server management, ITIL framework.
